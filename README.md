# ⚡ Electric Vehicle with Mild Regenerative Braking

## 📌 Overview

Designed and built a complete electric motorcycle prototype integrating mechanical design, embedded control, and energy-efficient regenerative braking.

This project focuses on real-world EV system development, combining chassis design, power electronics, embedded logic, and system-level integration.

---

## 🚀 Key Features

* ⚙️ Complete EV system design (mechanical + electrical)
* 🔋 Battery, motor controller, and wiring integration
* 🔄 Mild regenerative braking using BLDC motor control
* 🧠 Embedded control logic for real-time operation
* 📊 Structural validation using FEA (ANSYS)
* 🛠 Full prototype build and testing

---

## 🧠 Embedded & Control System

* PWM-based BLDC motor control
* ADC-based feedback for dynamic braking response
* Real-time system behavior handling
* Basic ECU-style control logic

---

## 🔌 Smart System Architecture

* Multi-ECU architecture for diagnostics and monitoring
* I2C communication between subsystems
* RS485-based BMS data acquisition
* Master controller for centralized data logging
* Fault detection and logging system (CSV-based)
* HTTP-based monitoring interface

---

## 📊 System Architecture Overview

* Master Controller + Multiple ECUs
* Communication Protocols:

  * I2C (inter-ECU communication)
  * RS485 (BMS interface)
  * HTTP (monitoring system)
* Sensors:

  * Hall Effect Sensor (motor RPM)
  * Battery Management System (BMS)

---

## 🏗 Mechanical Design

* Chassis Type: Dual cradle
* Material: AISI 4130 (C.S. seamless pipe)
* Designed using CATIA & Fusion 360
* Structural validation using ANSYS

### 📐 Key Specifications

* Total Weight: ~132 kg
* Wheelbase: 57 inches
* Rake Angle: 23°
* Frame optimized for strength-to-weight ratio

---

## 📈 Performance & Analysis

* Impact Load Tested: 6666 N
* Factor of Safety:

  * Front Impact: 7.83
  * Rear Impact: 7.85
  * Side Impact: 5.09
* Maximum Stress: ~108 MPa
* Deformation: ~0.22 mm

### 🔋 Braking Performance

* Regenerative + Hydraulic Disc Braking
* Braking Energy: ~6020 J
* Improved energy efficiency and control

---

## 🛑 Braking System

* Hydraulic disc braking system (KTM Duke-based components)
* High stopping power and heat dissipation
* Designed using friction and torque calculations
* Integrated with regenerative braking logic

---
